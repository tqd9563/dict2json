## Introduction

将python dict转换为标准json格式

## Usage
### Python字典转json
选中python dict，命令面板输入`Dict to JSON: Convert Python Dict to JSON`，或快捷键`Ctrl + Alt + J`（mac电脑为`Cmd + Option + J`）

### Json格式化
选中json，命令面板输入`Dict to JSON: Format JSON`，或快捷键`Ctrl + Alt + M`（mac电脑为`Cmd + Option + M`）

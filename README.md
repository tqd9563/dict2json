## Introduction

将python dict转换为标准json格式

## Usage
选中python dict，`cmd+shift+p`打开命令面板，输入命令`Dict to JSON: Convert Python Dict to JSON`即可

## Feature
- [x] 能够处理复杂的转义字符串
